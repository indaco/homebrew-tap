## sley

A CLI tool for managing semantic versioning using a .version file.

**GitHub:** <https://github.com/indaco/sley>

### Install

```bash
brew install indaco/tap/sley
```

### Upgrade

```bash
brew upgrade indaco/tap/sley
```

### Uninstall

```bash
brew uninstall sley
```

---

## Remove the Tap

```bash
brew untap indaco/tap
```
